Counter App — это простое графическое приложение на Java, разработанное с использованием Swing.

Программа представляет собой счётчик нажатий. При каждом нажатии на кнопку значение увеличивается на 1, а текст на экране обновляется. Также предусмотрена кнопка для сброса счётчика.

В приложении используются основные компоненты Swing:
JFrame — окно приложения
JLabel — отображение текста
JButton — кнопки для взаимодействия

Программа работает на основе обработки событий, реагируя на действия пользователя (нажатие кнопок).

Функциональность:

открывается окно приложения
отображается текущий счётчик
кнопка увеличивает значение
кнопка сбрасывает значение
